Trabajo de investigacion sobre la web

Abstract

Web es un sistema de documentos de hipertexto (se entiende por esto, la mezcla de textos
gráficos y archivos de todo tipo, en un mismo documento) a los que se puede acceder
mediante Internet.

Web e Internet no son sinónimos, pero si están relacionados ya que la web funciona a
través de Internet, que es un sistema de ordenadores que conforman una red de
comunicación. Desde la creación de la primera red ARPANet, la web no ha dejado de
cambiar y perfeccionarse.

En este trabajo se ha recurrido a diversas páginas webs con el fin de definir dicha evolución.
La primera etapa de desarrollo, la web 1.0 estaba compuestas por páginas web estáticas
conectadas sin contenido interactivo, después vendría la web 2.0 cuya principal diferencia
con la anterior es que en esta ya se puede participar, dicha web evoluciona hacia la web 3.0
cuyos sitios web son capaces de conectarse entre sí según los intereses del usuario.

Finalmente se profundiza en las aplicaciones características de la WEB 2.0, concretamente
de las aplicaciones sociales, las cuáles han cambiado la forma de comprender las
comunicaciones. Entre ellas se encuentra Facebook, que es una de las más populares
debido a que resulta ideal a la hora de promocionar productos, servicios o contenidos.
