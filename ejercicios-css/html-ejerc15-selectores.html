Tipos de selectores que entran en examen

  1. Clase (class=””)

    2. Selecciona los elementos que lleven class="algo" para que compartan un mismo estilo.

    Ejemplo:Se lo he puesto a los titulos 1, 3, y 5.

  2. Identificador (id=””)

    3. Selecciona un elemento que lleve id="algo" para darle a ese elemento un estilo diferente

    Ejemplo:A este parrafo le he puesto un id para que tenga otro estilo

  3. Todos los elementos (*)

    4. Selecciona todos los elementos

    Ejemplo: lo podriamos utilizar para poner el mismo tipo de letra a todo el documento.

  4. Varios elementos ( p, em, h1, h2)

    p sirve para hacer parrafos.
    Ejemplo: lo he empleado para los parrafos de este documento

    em sirve para poner encursiva
    Ejemplo: se lo he puesto a esta frase

    h1 sirve para poner el titulo principal
    Ejemplo: lo he utilizado en el titulo de este documento

    h2 sirve para poner titulos secundarios
    Ejemplo: lo llevan los nombres de los distintos conectores.

  5. Elementos contenidos en otros elementos (ol li)

    6. ol sirve para hacer una lista ordenada de elementos y li sirve para seleccionar los elementos de la lista

    Ejemplo: en este documento lo he empleado para hacer la lista de los conectores.

  6. Sobrevuleo (p:hover)

    7. Selecciona los elementos que queremos que cambien de estilo cuando se pase el raton por encima.

    Ejemplo: lo he empleado en este parrafo, al pasar el raton por encima cambia de estilo.
